# Explicação Detalhada

Desde Go 1.18, `any` é um apelido para `interface{}`.

Ele representa qualquer tipo.

É muito utilizado em APIs genéricas e bibliotecas.

Apesar de útil, seu uso excessivo elimina parte da segurança de tipos da linguagem.

# Saída Esperada

42

Go

true

# Código

```go
package main

import "fmt"

func imprimir(valor any) {
	fmt.Println(valor)
}

func main() {
	imprimir(42)
	imprimir("Go")
	imprimir(true)
}
```
